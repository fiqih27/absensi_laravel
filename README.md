## ⚡ TECH STACK & DEPENDENCIES
Sistem ini dibangun di atas infrastruktur modern untuk memastikan kecepatan transmisi data absensi secara *real-time*.

| Package / Extension | Purpose |
| :--- | :--- |
| **Laravel Framework** | Core System |
| **Guzzle HTTP** | API Communication |
| **Predis** | Redis Client for Speed |
| **ext-sockets** | Real-time Communication |
| **ext-pdo_mysql** | Database Engine |
